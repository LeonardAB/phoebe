# Part-Of-Speech Codes
This file outlines the Part-of-Speech opcodes generated by phoebe. The result is a BLOB column containing one or more of these opcodes as bytes. These are collected from all of the entry's `keb` and `reb` elements, so be warned, they may not apply to all readings or spellings of the given word.

For example, the word `為る・する` has in its POS column the data `49 1D E9 76 F0`.
This indicates that it is an irregular suru verb `0x49`, an intransitive verb `0x1D`, typically a suffix `0xE9`, an auxiliary verb `0x76`, and is usually written using kana alone `0xF0`.

## Querying the Database
To find all words that are nouns in the database, you can execute a query like this:

```sql
SELECT * FROM jisho WHERE instr(pos, x'00')
```
